## Run Locally

```bash
# Clone the repository
git clone https://github.com/AmaedaQ/portfolio.git

# Navigate to the project folder
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
````

Visit  [http://localhost:3000](http://localhost:3000)
